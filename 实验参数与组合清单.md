## 七、关键结论（答辩直接用）

### 7.1 方法选择

| 需求 | 推荐 | 理由 |
|------|------|------|
| 🏆 **全局最优** | **RL iter=5** | SSIM 0.963（最高）、Edge 0.64（最低振铃）、TV 1.25× |
| ⚡ **最快+好** | **Wiener K=0.03** | 9s 完成，SSIM 0.922，Edge 0.56 |
| 🛡️ **最保守** | **TV-L2 lam=0.002** | TV 1.25×（最低）、Edge 0.59 |
| ❌ **不推荐** | **任何含 psf-sigma 的组合** | 所有情况都加剧振铃 |

### 7.2 RL 迭代趋势（跨分辨率一致）

```
迭代 5 → 20 ：
  Laplacian ↑（+9 → +67 高分辨率 / +547 → +2238 低分辨率）
  TV ↑（1.20 → 1.60 / 1.25 → 1.73）
  SSIM ↓（0.947 → 0.908 / 0.963 → 0.930）
  Edge Ratio ↓（0.63 → 0.59 / 0.64 → 0.56）

结论：RL iter=5 在 320×240 和 640×480 上都是最佳平衡点。
```

### 7.3 核心发现

1. **K 曲线完整**：K=0.0001→1.0，Laplacian Δ 从 +35633 到 -345，SSIM 从 0.66 到 0.97 → 清晰的权衡关系
2. **鬼影根源**：静态帧（du≈0）Edge Ratio=0.94、SSIM=1.0 → 鬼影来自 PSF 运动估计偏差，非反卷积缺陷
3. **高分辨率难**：episode_0002（640×480）所有方法 TV>2×，SSIM<0.86 → 分辨率越高，对 PSF 误差越敏感
4. **参数泛化好**：RL iter=10 在所有旧格式上 SSIM>0.92，无需针对每个 episode 调参
5. **Adaptive K 有条件**：旧格式有效（+25-38%），新格式无效（-75%）
6. **PSF 误差敏感**：depth/exposure 偏离真值 → 重影系统性增加
7. **RL 趋势一致**：两种分辨率下迭代数与锐度/振铃的关系完全一致 → 结论可推广
