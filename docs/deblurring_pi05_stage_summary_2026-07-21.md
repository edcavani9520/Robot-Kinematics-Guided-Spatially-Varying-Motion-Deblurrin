# 运动学引导去模糊与 Pi0.5 下游任务：阶段性总结

**任务：** 将方块放入碗中。  
**状态：** 阶段性结果；下游试验尚未完成 2×2 全因子组合，且本轮未登记各组试验次数。

## 技术结论

在目前三组 Pi0.5 LoRA 微调结果中，基线（原始图像训练、原始图像推理）的成功率为 **54%**。以去模糊图训练、原始图像推理时为 **59%**，较基线高 **5 个百分点**；以原始图像训练、实时去模糊图推理时为 **57%**，较基线高 **3 个百分点**。这说明运动学引导去模糊在该单一桌面操作任务上呈现正向、但幅度有限的下游收益。

现有结果支持“去模糊值得保留为训练与部署候选预处理”的工程判断，但尚不足以证明统计显著性、确定最佳部署策略，或声称去模糊对成功率存在因果提升。原因是第四组尚未运行，且当前没有各组试验次数、随机化顺序和失败类型记录。

## 下游结果：三组已完成比较

| 组别 | LoRA 训练图像 | 部署时输入 | 成功率 | 相对原始基线的差异 |
|---|---|---|---:|---:|
| G1（基线） | 原始图像 | 原始图像 | 54% | 0 个百分点 |
| G2（推理预处理） | 原始图像 | 实时去模糊图像 | 57% | +3 个百分点 |
| G3（训练预处理） | 去模糊图像 | 原始图像 | 59% | +5 个百分点 |
| G4（联合预处理） | 去模糊图像 | 实时去模糊图像 | 未测试 | — |

其中，G2 的 **57%** 对应“**原始图像训练 + 实时去模糊推理**”，而非去模糊训练模型。

### 可作出的描述性解释

- **训练侧收益更大：** 在现有点估计中，G3 比 G1 高 5 个百分点（相对提升约 9.3%），表明去模糊训练数据可能让策略学习到更稳定的视觉表征。
- **部署侧也有正向收益：** G2 比 G1 高 3 个百分点（相对提升约 5.6%），说明不重新训练时，在原始训练策略前加入实时去模糊也可能有帮助。
- **不能比较两种机制的强弱：** G3 比 G2 高 2 个百分点，但二者改变的是不同因素；没有共同的试验次数、随机化与置信区间，不能据此断言“训练去模糊显著优于实时推理去模糊”。
- **G4 是当前最关键的缺口：** 它用于估计训练输入与部署输入的一致性，以及二者叠加后是否继续增益、饱和或因伪影/延迟而退化。

## 图像侧实验与最终运行配置

离线消融采用 5 段当前格式 H5 轨迹（共 528 帧）进行。最终供下游实验使用的保守配置为：

| 项目 | 设定 |
|---|---|
| 去模糊器 | Wiener |
| 正则参数 K | 0.03 |
| 深度 | 0.5 m |
| 曝光时间 | 0.01 s |
| 内参 | fx = fy = 260.645 px；cx = 159.5 px；cy = 119.5 px |
| PSF 平滑 | 关闭 |
| 自适应 K | 关闭 |

此前记录的留出轨迹汇总显示，Wiener（K=0.03）相对输入模糊图的 Laplacian 能量约提高 **50.4%**、Tenengrad 约提高 **9.2%**，同时 SSIM 约为 **0.9992**；RL（5 次迭代）得到相近的清晰度指标，但离线批处理速度更低。TV-L2（λ=0.001）更保守，清晰度增益较小而结构保留更强。因此为降低伪影风险并兼顾处理速度，当前下游实验固定采用 Wiener K=0.03。

参数消融也给出两个明确的负面结论：

- 高强度 PSF 平滑会显著放大高频指标，同时损害边缘比与 SSIM，属于明显伪影风险，已关闭。
- 自适应 K 只带来很小的清晰度增益，却略损害保真指标并降低速度，已关闭。

## 指标定义与解释边界

图像侧没有无模糊真值。因此 PSNR/SSIM 是“输出相对输入模糊图”的保真度量，而非真实复原精度；Laplacian、Tenengrad 和总变分描述高频变化，也可能奖励振铃或噪声。它们仅用于选择一个保守、可部署的参数区间，不能单独证明图像恢复了真实细节。

下游成功率是任务级最终指标，但本轮没有每组试验次数。因而不能计算 Wilson 置信区间、二项检验或 Fisher 精确检验，也不能排除物体初始位置、光照、操作者节奏和试验顺序造成的差异。离线批处理 FPS 包含 PSF 构建和评价开销，不能等同于 Pi0.5 部署端到端控制延迟。

## 系统适用范围

该方法以机器人关节位姿和相机—末端近似关系估计曝光内像素运动，采用全局线性 PSF。它适用于小旋转、窄视场、场景深度变化有限、相机外参近似稳定的桌面任务。当前仍存在以下建模限制：

- 固定深度与近似内参，而非逐帧深度/严格标定；
- 有限差分关节速度与全局 PSF，未显式建模滚动快门、空间变化模糊或遮挡；
- 曝光时间为当前工作假设，真实相机曝光若偏离会直接改变 PSF 尺度；
- Python API 虽已补充 CLI 边界约束，仍应以导出的最终 H5、训练图像和实时输入路径保持同一配置，避免训练—部署域不一致。
